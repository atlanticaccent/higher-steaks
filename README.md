# Higher Steaks

Adds a new "Tartare" Stake, which applies all modded Stakes by
default, on top of all vanilla Stakes.

Which modded Stakes are applied by Tartare Stake can be changed in
the mod's config menu.

Bonus: If Tartare Stake is configured to _not_ apply a modded Stake,
that Stake will never be applied to runs on Tartare Stake even if
Tartare Stake is also configured to apply another modded Stake that
would apply it.

## Known Issues

- The "also applies" section for Tartare Stake on the new run screen
  may say "Gold Stake" followed by "nil" repeatedly
- Tartare Stake may not appear in the expected position in the Stake
  selector of the new run screen
- The "Run Info -> Stake" screen may not display all modded Stakes
  applied by Tartare Stake (this is a pre-existing vanilla/SMODS-
  recreation-of-vanilla limitation)
