# Higher Steaks

Adds a new "Tartare" Stake, which applies all modded Stakes by default, on top of all vanilla Stakes.

Which modded Stakes are applied by Tartare Stake can be changed in the mod's config menu.

## WIP

Bonus: If Tartare Stake is configured to _not_ apply a modded Stake applied by a another modded Stake it _is_ applying, then the "disabled" Stake (should) be prevented from applying on runs on Tartare Stake whilst still applying the Stake that would have applied it.
